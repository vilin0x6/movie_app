Movie App
=========
Movie App은 영화의 짧은 소개글을 보여주는 웹 앱입니다. YTS torrents에서 제공하는 Web API를 통해 영화 차트 정보를 실시간으로 가져오며, 매일 업데이트 되는 영화 리스트를 통해 오늘 볼 영화를 쉽게 고를 수 있습니다.

React를 이용해 화면을 구성했습니다.
